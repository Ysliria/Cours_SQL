Installation

Sur Windows

Il existe plusieurs méthodes pour installer MySQL sur Windows.
La première consisterait à aller sur le site de MySQL et à télécharger la version compatible avec Windows. Ensuite il vous faudrait suivre pas à pas la documentation pour effectuer l'installation du SGBDR et de ses dépendances.

Une autre méthode consiste à télécharger un serveur de développement comme WAMP ou Laragon. Nous allons partir sur cette méthode.

  1. Téléchargez l'un des serveur de développement
  2. Installez-le
  3. Une fois installé, démarrez-le
  4. Via votre invite de commandes, déplacez-vous dans son répertoire d'installation :
    • Pour Laragon : cd \laragon\bin\mysql\mysql-5.7.24-winx64\bin\
    • Pour WAMP : cd \wamp64\bin\mysql\mysql-5.7.24-winx64\bin\
  5. Enfin saisissez mysql --version

Vous devriez voir s'afficher : 


    λ mysql --version
    mysql  Ver 14.14 Distrib 5.7.24, for Win64 (x86_64)

Sur MAC

De la même manière sur MAC on pourrait télécharger directement sur le site de MySQL la bonne version et suivre les instructions.
Mais nous allons installer MAMP.

  1. Téléchargez MAMP
  2. Installez-le
  3. Une fois installé, démarrez-le
  4. Dans le terminal allez dans cd /Applications/MAMP/Library/bin
  5. Exécutez ./mysql --version

Vous devriez voir s'afficher : 


    % ./mysql --version
    ./mysql  Ver 14.14 Distrib 5.7.32, for osx10.12 (x86_64) using Editline wrapper
ATTENTION ! N'installez jamais MAMP sur Windows !
