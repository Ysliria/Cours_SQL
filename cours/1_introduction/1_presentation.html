Présentation

Le SQL

Le SQL (Structured Query Language) est un langage permettant de communiquer avec une base de données. Ce langage est notamment utilisé pour communiquer avec les données d’un site web. C'est le langage pour base de données le plus répandu.

Attention : Il ne faut pas confondre le langage SQL et le SGBDR MySQL !

La base de donnnées

Une base de données informatique est un ensemble de données qui ont été stockées sur un support informatique, organisées et structurées de manière à pouvoir facilement consulter et modifier leur contenu. Le SQL nou permettra de gérer et interagir avec cette base ; afin de pouvoir ajouter, modifier, supprimer ou tout simplement afficher des éléments de la base.
Une base de données seule ne suffit donc pas, il est nécessaire d'avoir également :

  • un système permettant de gérer cette base (le SQL),
  • un langage pour transmettre des instructions à la base de données (par l'intermédiaire du système de gestion).

SGBDR

Un Système de Gestion de Base de Données (SGBD) est un logiciel (ou un ensemble de logiciels) permettant de manipuler les données d'une base de données, c'est-à-dire sélectionner et afficher des informations tirées de cette base, modifier des données, en ajouter ou en supprimer (ce groupe de quatre opérations étant souvent appelé "CRUD", pour Create, Read, Update, Delete). MySQL est un système de gestion de bases de données.

Le R de SGBDR signifie "relationnel". Un SGBDR est un SGBD qui implémente la théorie relationnelle. MySQL implémente la théorie relationnelle ; c'est donc un SGBDR.
Nous verrons les relations plus tard dans le cours !
