Installation de MySQL Workbench

MySQL Workbench

MySQL Workbench est un logiciel de gestion et d'administration de bases de données MySQL créé en 2004.
Via une interface graphique, il permet, entre autres, de créer, modifier ou supprimer des tables, des comptes utilisateurs, et d'effectuer toutes les opérations inhérentes à la gestion d'une base de données. Pour ce faire, il doit être connecté à un serveur MySQL.

Installation

Allez sur le site dédié à MySQL Workbench et téléchargez la version en fonction de votre système d'exploitation.
Installez-le et démarrez-le. Vous devriez arriver sur l'accueil comme suit :

Configurations

Nous allons configurer 2 choses, la connexion à la base de données et la version de la BDD que nous allons utiliser.
Pour la version de la BDD, allez dans Edit/Preferences..., et dans l'onglet Modeling/MySQL verifier la version de votre BDD (5.7.xx).

Ensuite nous allons configurer la connexion à la base de donnée.

  1. Sur la page d'accueil, cliquez sur le petit plus à côté de MySQL Connections.
  2. Saisissez un nom pour identifier la connexion.
  3. Saisisser les informations de connexion (identifiant et mot de passe).
  4. Attention à bien vérifier le port utilisé par la BDD ; sur Windows c'est bien du 3306, mais sous MAC il se peut que le port par défaut soit différent (contrôler votre serveur MAMP). Si vous testez la connexion vous devriez vite savoir si le paramétrage est bon.
  5. Sauvegardez.

Une fois terminé, redémmarrez MySQL Workbench pour être sûr que les paramétrages ont bien été pris en compte.
Allez ensuite dans l'onglet Models.
